# Color Controller
This is a color picker which returns a color code in hex format. It shows in the side bar under the `properties` tab. This returns a `Color` object.

```tsx
type Color = {
  colorHex: string;
  colorName: string;
  id: string;
};
```

to render the color in your component,  call the `getColor` function from `visio-cms-lib/utils` and pass the `Color` object as the argument.

This function makes sure the color is rendered correctly when it's reference from a color scheme in your themes tab.
