# Link Controller

The Link Controller enables content editors to link pages or external URLs to your components. It appears in the sidebar under the `properties` tab.

To retrieve the correct URL of the selected page, use the `getLink` function from `visio-cms-lib/utils`, passing the `link value` as the argument. This function returns a valid URL with the active locale appended to it. Additionally, when the project is in builder mode, it returns `#`. If a linked page's slug changes, the function automatically updates the link with the new slug, ensuring there are no broken links in your project.
