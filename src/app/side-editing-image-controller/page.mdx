# Color Controller
This an image chooser which opens the media libray. Editors can upload images, crop and choose from unsplash. It shows in the side bar under the `properties` tab.
this returns a `MediaFile` object.

```tsx
type MediaFile = {
  mediaHash: string | undefined;
  altText: string;
  width: number;
  height: number;
}
```

To get the correct url of the image, call the `getImageUrl` function from `visio-cms-lib/utils` and pass the `MediaFile` object as the argument.
