# getImgeUrl
Similar to the `getColor` function, the `getImageUrl` function is used with the `MediaFile` type and the `image` controller to get an image url. The function returns the `url` of the image if the `mediaHash` is found in your media explorer. If the `mediaHash` is a url to an image, the `mediaHash` is returned.  


```jsx  

import { getImageUrl } from 'visio-cms-lib/utitls';
import {Block, MediaFile} from 'visio-cms-lib/types';

type Props = {
  image: MediaFile;
}

const Hero:Block<Props> = ({image}) => {
  return (
    <img src={getImageUrl(image)} width={image.width} height={image.height} alt={image.altText}/>
  )
}

ColorComponent.Schema = {
   ....,
   defaultValues:{
     color:{
        mediaHash: 'https://www.example.com/image.jpg', //could also be an id of an image in your media explorer
        altText: 'This is an image',
        width: 100,
        height: 100,
     }
   },
   sideEditingProps: [
    {
      propName: 'image',
      type: 'image',
      label: 'Hero Image',
    },
   ]
}


```
