# Text Controller
Similar to in page text controller, this controls a `string` but shows in the side bar under the `properties` tab. Very important if you want to pass some custom text data to your component. eg. `classNames`, `styles`, `color codes` etc.

<Row>
  <Col>

      ### keys

    <Properties>
      <Property name="propName" type="string">
        This represents the prop this controller is controlling.
      </Property>

       <Property name="type" type="SideEditingProps">
        This represents the type of the controller. In this case it should be `text`.
      </Property>

      <Property name="label" type="string">
        This represents the label of the controller input.
      </Property>
    </Properties>

  </Col>
  <Col sticky>

    ```jsx {{title: 'Schema'}}
       {
        ...,
        sideEditingProps: [
            {
                propName: 'propName',
                type: 'text',
                label: 'Enter class Name'
            }
        ]
       }
    ```

  </Col>
</Row>
