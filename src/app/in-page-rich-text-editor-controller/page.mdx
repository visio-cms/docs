# Rich Text Editor Controller
Similar to the text controller, the rich text editor features an editor that allows content editors to directly modify text on the page and manage the value of a text prop within a block. The value of the rich text is in HTML format, supporting text formatting options like bold, italic, and more.
It is imported from visio-cms-lib/RichTextEditor and integrated into the block component to display the text.

<Row>
  <Col>

    ### Props

    <Properties>
      <Property name="propName" type="string">
        This represents the prop this controller is controlling.
      </Property>

      <Property name="defaultValue" type="string">
        The default value of the text input field. This is normally defined in the `defaultPropValues` object of the block's `schema`.
      </Property>

     <Property name="allowedControls" type="EditorControlTypes[]">
        This is a list of formating options that you want to allow in the editor. The options are:
        - bold
        - italic
        - strike
        - underline
        - paragraph
        - h1
        - h2
        - h3
        - h4
        - h5
        - h6
        - bullet-list
        - ordered-list
        - blockquote
        - code
        - link
        - image
        - text-color
        - background-color
        - align-center
        - align-right
        - align-left
        - align-justify

     The rich text editor comes with all these controls by default.
     `Note` To avoid [hydration](https://nextjs.org/docs/messages/react-hydration-error) issues, only wrap the `RichTextEditor`  in a `div` or `section` tags

    </Property>

     <Property name="allowNewLines" type="boolean">
        Only needed if you want to accept line breaks in the editor. This is set to `true` by default.
      </Property>
    </Properties>

  </Col>
  <Col sticky>

    ```jsx
       <RichTextEditor
        allowNewLines={false}
        allowedControls={['bold', 'italic', 'underline', 'h1']}
        propName={'propName'}
        pageBlockId={pageBlockId}
        defaultValue={propName}
        />
    ```

  </Col>
</Row>
